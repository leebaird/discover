# Wordlists

## Commands

Built-in:
    /usr/share/wordlists/rockyou.txt
    /usr/share/wordlists/seclists/Usernames/
    /usr/share/wordlists/seclists/Passwords/

Scrape from a web page:
    cewl "http://<target IP>/joomla/" -w passwords.txt

Mutate a base word with crunch:
    crunch 7 7 -t apple^% -o passwords.txt

Combine users and passwords for spraying:
    # users.txt — one username per line
    # passwords.txt — one password per line

## Validation / False Positives

Site-specific cewl output beats raw rockyou for targeted spraying — still validate policy lockouts before large sprays.


# Hash Identification

## Commands

hashid <hash>
hashcat --example-hashes | grep -i ntlm

Common modes:
    0      MD5
    100    SHA1
    400    phpass (WordPress, Joomla)
    500    md5crypt (Unix)
    1000   NTLM
    1500   descrypt (Unix DES)
    1600   Apache $apr1$
    1800   sha512crypt (Unix)
    5600   NetNTLMv2
    13100  Kerberoast TGS-REP
    18200  Kerberos AS-REP

Kerberoast and AS-REP collection: see ad-attacks.txt


# john

## Commands

NTLM / NetNTLM:
    john --format=NT --wordlist=/usr/share/wordlists/rockyou.txt hashes.txt

Linux shadow (after unshadow):
    unshadow passwd shadow > combined.txt
    john --wordlist=/usr/share/wordlists/rockyou.txt combined.txt

Zip archive:
    zip2john test.zip > zip.hash
    john --wordlist=/usr/share/wordlists/rockyou.txt zip.hash

Show cracked passwords:
    john --show hashes.txt

Hash file format examples:
    user:hash
    $NT$hash

## Validation / False Positives

john --show with zero cracked lines means the wordlist or format is wrong — re-check with hashid.


# hashcat

## Commands

Basic attacks:
    hashcat -m 1000 hashes.txt /usr/share/wordlists/rockyou.txt
    hashcat -m 5600 hashes.txt /usr/share/wordlists/rockyou.txt
    hashcat -m 13100 kerberoast.txt /usr/share/wordlists/rockyou.txt
    hashcat -m 18200 asrep.txt /usr/share/wordlists/rockyou.txt

With rules:
    hashcat -m 1000 hashes.txt /usr/share/wordlists/rockyou.txt -r /usr/share/hashcat/rules/best64.rule

Show cracked hashes:
    hashcat -m 1000 hashes.txt /usr/share/wordlists/rockyou.txt --show

Resume interrupted session:
    hashcat --session crack1 -m 1000 hashes.txt /usr/share/wordlists/rockyou.txt
    hashcat --restore

Common reference:
    -m 0     MD5
    -m 400   WordPress / Joomla phpass
    -m 1000  NTLM
    -m 1800  Unix SHA-512 crypt
    -m 5600  NetNTLMv2

## Validation / False Positives

STATUS exhaustED means the wordlist finished with no match — try rules, a larger list, or targeted cewl output.
GPU errors on VMs may need `--force` — confirm cracked output with --show before reporting.


# Linux Shadow Hashes

## Commands

Copy /etc/passwd and /etc/shadow from target (SSH, meterpreter, or LFI).
    unshadow passwd shadow > combined.txt
    john --wordlist=/usr/share/wordlists/rockyou.txt combined.txt
    hashcat -m 1800 combined.txt /usr/share/wordlists/rockyou.txt

## Validation / False Positives

Cracked root access still requires a shell or SSH key — confirm login path separately.

# Online Attacks

## Overview

Brute force and password spraying against network services. Prefer spraying with lockout awareness — see ldap-owa.txt for OWA/LDAP workflows.

## Commands

hydra:
    hydra -L users.txt -P passwords.txt -M targets.txt <protocol>
    hydra -l <user> -P passwords.txt <target IP> ssh
    hydra -l <user> -P passwords.txt <target IP> http-post-form "/login:user=^USER^&pass=^PASS^:F=incorrect"

crackmapexec spray:
    crackmapexec smb <target IP> -u users.txt -p 'Spring2026!' --continue-on-success
    crackmapexec smb <target IP> -u users.txt -p passwords.txt --continue-on-success

medusa:
    medusa -h <target IP> -u <user> -P passwords.txt -M ftp

ncrack:
    ncrack -vv --user <user> -P passwords.txt rdp://<target IP>

## Validation / False Positives

A single valid login from spraying is high impact — confirm with a second action (share listing, whoami over winrm).
Document lockout policy and stop after the agreed spray count.


# Captured NetNTLM Hashes

## Overview

After Responder or other capture (see ad-attacks.txt), crack NetNTLMv2 offline.

## Commands

hashcat -m 5600 responder_hashes.txt /usr/share/wordlists/rockyou.txt
john --format=netntlmv2 --wordlist=/usr/share/wordlists/rockyou.txt responder_hashes.txt

## Validation / False Positives

Cracked NetNTLMv2 gives the challenge response password — verify with crackmapexec or evil-winrm before reporting domain access.
Relay without cracking is often better when SMB signing is off — see ad-attacks.txt.
