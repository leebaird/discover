Ubuntu

- Download
https://ubuntu.com/download/desktop

- Patch OS
Right-click on the Desktop > Open Terminal
sudo apt update; sudo apt -y upgrade; sudo apt -y dist-upgrade; sudo apt -y autoremove; sudo apt -y autoclean

- Install basic tools
sudo apt install -y curl git net-tools open-vm-tools-desktop plocate python3-pip whois
sudo snap install --classic code (Intel only)

- Install Go
Check for the latest version at https://go.dev/doc/install
Choose the correct architecture.
wget https://go.dev/dl/go1.25.6.linux-arm64.tar.gz
sudo tar -C /usr/local -xzf go1.25.6.linux-arm64.tar.gz
rm go1.25.6.linux-arm64.tar.gz
open .bashrc
PATH=$PATH:/usr/local/go/bin
source .bashrc
go version

- Install uv
sudo curl -LsSf https://astral.sh/uv/install.sh | sh
Close Terminal and reopen.
uv

- Patch OS again and update locate database
sudo apt update; sudo apt -y upgrade; sudo apt -y dist-upgrade; sudo apt -y autoremove; sudo apt -y autoclean; sudo updatedb

- Ignore case for tab completion
sudo open /etc/inputrc
Uncomment: set completion-ignore-case On
Save the file and close.
Close Terminal and reopen.

- Clean up home folder
rm -rf Documents/ Music/ Pictures/ Public/ Templates/ Videos/

- Install Discover
    cd ~/
    sudo git clone https://github.com/leebaird/discover.git
    cd discover/config/
    ./install.sh
    Exit Terminal and reopen
    Test your network connection with 'n'

Take a snapshot of your VM.

    cd discover/

- Fix Burp Suite Pro browser permissions
find / -name "chrome-sandbox" 2>/dev/null | grep Burp
sudo chown root:root /path/to/chrome-sandbox
sudo chmod 4755 /path/to/chrome-sandbox
