## Commands

    sudo apt update; sudo apt -y upgrade; sudo apt -y dist-upgrade; sudo apt -y autoremove; sudo apt -y autoclean


# Basic Tools

## Commands

    sudo apt install -y curl git golang net-tools open-vm-tools-desktop plocate python3-pip whois
    sudo snap install --classic code


# uv

## Overview

Install uv for fast Python tooling (PEP 668-safe workflows).

## Commands

    sudo curl -LsSf https://astral.sh/uv/install.sh | sh

Close Terminal and reopen, then verify:

    uv

Patch again and refresh locate:

    sudo apt update; sudo apt -y upgrade; sudo apt -y dist-upgrade; sudo apt -y autoremove; sudo apt -y autoclean; sudo updatedb


# Desktop Tuning

## Tab completion

    sudo gedit /etc/inputrc

Uncomment: `set completion-ignore-case On`
Save, close Terminal, and reopen.

## Home folder cleanup

    rm -rf Documents/ Music/ Pictures/ Public/ Templates/ Videos/

## Power settings

Settings (top-right cog) > Power > Power Saving > Automatic Screen Blank: off

## Text editor (gedit)

    touch tmp && gedit tmp

Preferences:
    Display Line Numbers: on
    Highlight Current Line: on
    Show Right Margin: on
    Margin Position: 120
    Auto Indentation: off
    Character: Space
    Spaces Per Tab: 4
    Spaces Per Indent: 4
    Restore Session: off

Close gedit and delete tmp.


# Discover Install

## Commands

    cd ~/
    sudo git clone https://github.com/leebaird/discover.git
    cd discover/config/
    ./install.sh

Exit Terminal and reopen.
Test network with Discover option that runs connection check, or `./discover.sh` and verify connectivity.

Take a VM snapshot before further customization.


# Discover Update

## Commands

    cd ~/discover/
    ./discover.sh

Option 18 (Update) — installs tool updates via misc/update.sh and rebuilds notes HTML.


# Scanner User Agents

## Overview

Discover Update refreshes a Microsoft Edge User-Agent into
`resource/user-agent.txt` and applies it to Nikto, Nmap, and ffuf (`~/.config/ffuf/ffufrc`)
when those configs are writable. Active (httpx/whatweb) and other Discover scripts
read the same file via `f_discover_user_agent` / `$USER_AGENT`.

Manual override or inspection:

    cat /path/to/discover/resource/user-agent.txt

Reference list (human):
    https://www.whatismybrowser.com/guides/the-latest-user-agent/edge

JSON source used by Update:
    https://jnrbsn.github.io/user-agents/user-agents.json


# Burp Browser Sandbox

## Overview

Fix Burp Suite Pro embedded browser sandbox permissions on Linux.

## Commands

    find / -name "chrome-sandbox" 2>/dev/null | grep Burp
    sudo chown root:root /path/to/chrome-sandbox
    sudo chmod 4755 /path/to/chrome-sandbox


# Validation / False Positives

uv install requires a new shell before `uv` is on PATH; reopen Terminal if command not found.
Discover install.sh must finish before first `./discover.sh` run; missing config breaks menu options.
Burp chrome-sandbox path changes per Burp version; re-run find after upgrades.
