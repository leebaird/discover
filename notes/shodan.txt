# Setup

## Commands

Install CLI:
    pip install shodan
    shodan --help

API key from account.shodan.com, then:
    shodan init <API key>


# GUI Search Filters

## Overview

Use two-letter ISO country codes: https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2

Country          country:US
State            state:TN
City             city:Nashville
Organization     org:"ACME Corp"
Hostname         hostname:google
                 hostname:.nist.gov
                 hostname:.edu
Network          net:172.16.0.0/16
OS               os:"windows 7"
Port             port:21
                 -port:22
Product          product:openssh
Timeframe        before:31/12/2022 after:10/12/2022
Filters          all:comments
                 all:compromised
                 all:hacked


# GUI Dorks

## Overview

Quoted banner/title strings useful in GUI or CLI searches.

"Additional Interfaces"
"All Storage Devices" IPC$
"Authentication: disabled" port:"445"
"Cisco 3750"
"Cisco 7606"
"Original Siemens Equipment"

Example:
    country:mx city:"Playa del Carmen" port:445 all:comments


# CLI Query and Host Lookup

## Commands

shodan count port:22 country:US                          Count results for a filter
shodan count 'microsoft iis 6.0'                         Count multi-word product query
shodan count country:us state:tn city:nashville          Geo-filtered count
shodan count "All Storage Devices"                       Count dork matches
shodan count city:Nashville "Authentication: disabled"   Count SMB auth-disabled hosts

shodan search --fields ip_str,port,org,hostnames 'microsoft iis 6.0'
shodan search --fields location.country_code,ip_str,port,org,hostnames <search term>
shodan search org:<organization> country:<country code> <search term>
shodan search country:us state:tn city:nashville port:23
shodan search "All Storage Devices"
shodan search city:Nashville "Authentication: disabled" > smb.json
shodan search product:openssh -port:22                   OpenSSH not on default port 22

shodan host <target IP>                                  Host history and open ports
shodan domain <domain>                                   Subdomains and DNS for a domain


# CLI Download Parse Convert

## Commands

shodan download --limit 200 nginx.json.gz product:nginx
shodan download --limit -1 hacked.json.gz 'http.title:"hacked by"'
shodan download test.json.gz 'microsoft iis 6.0'

shodan parse --fields ip_str,product,title test.json.gz
shodan parse --fields title test.json.gz
shodan parse --fields ip_str,port,org --separator , test.json.gz
shodan parse test.json.gz                                Default fields from download

shodan convert <data file> <file format>
shodan convert nginx.json.gz xlsx
shodan convert remote-desktops.json.gz images


# CLI Stats and Facets

## Commands

shodan stats --facets country <service>                  Breakdown by country (e.g. apache)
shodan stats --facets country,org <search term>
shodan stats --facets org country:<country code> <search term>
shodan stats --facets port country:us
shodan stats --facets org port:3389 country:us
shodan stats --facets org country:us state:tn city:nashville
shodan stats --facets country "All Storage Devices"

shodan stats --facets org ssl.version:sslv2 http country:us
shodan stats --facets org country:us ssl.version:sslv2
shodan stats --facets ssl.version has_ssl:true http country:us
shodan stats --facets ssl.cert.fingerprint has_ssl:true http country:us
shodan stats --facets ssl.cert.fingerprint:100 has_ssl:true http country:us


# On-demand Scan

## Commands

shodan scan submit <target IP>                           Queue on-demand scan for one host
shodan scan submit --filename test.json.gz <CIDR>        Scan CIDR and save results
shodan parse test.json.gz                                  Parse scan output file


# Bulk Host Loop

## Overview

Query Shodan host history for a list of IPs. See bash.txt for loop patterns.

## Commands

#!/bin/bash
shodan init <API key>
echo "Gathering Shodan info on the following targets:"

for i in $(cat targets.txt); do
    echo $i
    shodan host $i >> shodan-hosts.txt
done


# Validation / False Positives

Shodan host/port data may be weeks or months old — verify with nmap or direct connection before reporting.
Free API tiers hit query and download credit limits — `count` is cheaper than bulk `download --limit -1`.
Multi-word CLI queries need quotes: `'microsoft iis 6.0'` not bare words.
`shodan host` shows historical banners; absence of a port in Shodan does not prove it is closed today.
GUI country codes are two letters; CLI often accepts lowercase `country:us` — both work in practice.
