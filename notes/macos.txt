macOS


# Overview

macOS assessment workstation setup: Homebrew, proxychains over SSH SOCKS, updog, and ~/.bash_aliases for navigation and quick HTTP serving. Default shell is zsh — source aliases from ~/.zshrc or ~/.bash_profile. For Linux equivalents, see linux.txt and web-burp.txt.

# Homebrew Setup

## Commands

Install:
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    brew install curl wget coreutils proxychains-ng

Troubleshooting:
    brew update                                      Run twice if the first pass errors
    brew doctor

## Validation / False Positives

`gls` in aliases requires coreutils — install via brew above or aliases fall back to errors.
Apple Silicon Homebrew prefix is /opt/homebrew; Intel uses /usr/local — adjust paths below if commands not found.


# Proxychains

## Overview

Dynamic SOCKS via SSH, then route tools through proxychains. Burp integration in web-burp.txt.

## Commands

Config (Intel: /usr/local/etc/proxychains.conf — Apple Silicon: /opt/homebrew/etc/proxychains.conf):
    nano /usr/local/etc/proxychains.conf
    socks5  127.0.0.1    1080                         Add to the bottom

SOCKS tunnel:
    ssh -D 1080 -f -C -q -N <user>@<target IP>

Test:
    proxychains4 curl http://<domain>

# Aliases

## Overview

Paste into ~/.bash_aliases and source from ~/.bash_profile, or copy into ~/.zshrc on default zsh. Preserves searchable one-liners.

## Commands

    vi ~/.bash_aliases

BLUE='\033[1;34m'
NC='\033[0m'

DNS=$(grep nameserver /etc/resolv.conf | awk '{print $2}')
EXTIP=$(curl ifconfig.co)
MYIP=$(ifconfig | grep broadcast | awk '{print $2}' | head -1)
MAC=$(ifconfig | grep ether | awk '{print $2}' | head -1)

alias c='clear'
alias cl='clear ; gls -l --color=auto'
alias cla='clear ; gls -la --color=auto'
alias d='cd $HOME/Desktop/ ; clear'
alias date='date +"%a %b %d, %Y - %r %Z"'
alias e='exit'
alias l='gls -l --color=auto'
alias la='gls -la --color=auto'

alias n='
echo ;
echo -e "${BLUE}Networking info${NC}"
echo -e "${BLUE}----------------------------------------${NC}"
echo -n "External IP:  "$EXTIP ; echo ;
echo -n "Internal IP:  "$MYIP ; echo ;
echo -n "MAC address:  "$MAC ; echo ;
echo -n "DNS:          "$DNS ; echo ;
echo ;
echo -e "${BLUE}Local connections${NC}"
echo -e "${BLUE}----------------------------------------${NC}"
netstat -ant | egrep "(Address|ESTABLISHED)" | egrep -v "(:|Flags|Nextref)" ;
echo  ;
echo -e "${BLUE}Verifying connectivity to Google${NC}"
echo -e "${BLUE}----------------------------------------${NC}"
ping -c3 google.com ; echo'

alias r='cd $HOME ; clear'
alias s='cd ~/tools/discover/ ; clear'
alias sip='sort -n -u -t . -k 1,1 -k 2,2 -k 3,3 -k 4,4'
alias ssh='ssh -o "XAuthLocation=/opt/X11/bin/xauth"'
alias t='cd ~/tools/'

alias update='
clear
echo
echo -e "${BLUE}[*] Updating Homebrew.${NC}"
brew update; brew upgrade
echo

if [ -d ~/tools/discover/.git ]; then
    echo -e "${BLUE}[*] Updating Discover.${NC}"
    cd ~/tools/discover/ && git pull
    echo
fi

echo -e "${BLUE}[*] Updating locate database.${NC}" ; echo
echo "Be patient, this takes a couple min." ; echo
cd $HOME
sudo /usr/libexec/locate.updatedb
echo'

alias upload='mkdir -p ~/Desktop/tmp; cd ~/Desktop/tmp/ ; updog -p8000 --ssl'
alias web='mkdir -p ~/Desktop/tmp; echo $MYIP ; cd $HOME/Desktop/tmp/ ; echo ; python3 -m http.server 80'
alias web2='mkdir -p ~/Desktop/tmp; echo $MYIP ; cd $HOME/Desktop/tmp/ ; echo ; python3 -m http.server 8000'


# Validation / False Positives

Aliases in ~/.bash_aliases do not load in zsh until sourced — copy to ~/.zshrc or add `[ -f ~/.bash_aliases ] && . ~/.bash_aliases` to ~/.zshrc.
`web` on port 80 usually requires sudo on macOS — use `web2` (8000) or `sudo` when binding to 80.
`ifconfig | grep broadcast` for MYIP is fragile on newer macOS — try `ipconfig getifaddr en0` if internal IP is blank.
EXTIP via ifconfig.co requires outbound HTTPS — fails on isolated lab networks.
X11 forwarding from macOS requires XQuartz — see linux.txt SSH X11 section.
