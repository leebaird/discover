# Download and Login

## Commands

https://www.kali.org/get-kali/#kali-virtual-machines
Select VMware Weekly (or your hypervisor format).

Verify hash:
    sha256sum /path/to/kali-image
    # Compare to checksum on the download page

Default login: kali:kali


# User Setup

## Commands

sudo useradd -m -s /usr/bin/zsh <NEW USER>
sudo passwd <NEW USER>

sudo visudo
# Add last line:
<NEW USER> ALL=(ALL) NOPASSWD:ALL

sudo reboot
Login as the new user.

Remove default account (optional, after confirming new user works):
    sudo userdel -r kali


# Desktop and Terminal

## Commands

Disable screen lock:
    Kali menu > Xfce Screensaver
    Screensaver > Enable Screensaver: off
    Lock Screen > Enable Lock Screen: off

Desktop icons:
    Right click Desktop > Desktop Settings > File/Launcher Icons
    Uncheck Home, File System, and Trash

Terminal font:
    Terminal > Preferences > Appearance > Font 11, transparency 0%

Shell prompt (optional):
    kali-tweaks > Shell & Prompt > Configure Prompt > One Line

Mousepad:
    Edit > Preferences
    View: Font 11, uncheck system monospace
    Editor: tab width 4, spaces, automatic indentation off

Timezone:
    sudo timedatectl set-timezone America/New_York
    # Central: America/Chicago, Mountain: America/Denver, Pacific: America/Los_Angeles

Patch OS:
    sudo apt update ; sudo apt -y upgrade ; sudo apt -y dist-upgrade ; sudo apt -y autoremove ; sudo apt -y autoclean ; sudo updatedb


# Install Discover

## Commands

cd ~
git clone https://github.com/leebaird/discover.git
cd discover/config/
./install.sh

cd ~
source ~/.zshrc
update

cd ~/discover/
./discover.sh
Option 18

## Validation / False Positives

The update alias is defined in Discover config — open a new shell after install.sh if update is not found.
Option 18 runs misc/update.sh; confirm key tools with which nmap ffuf gowitness.

# Cloak System

## Overview

Optional OPSEC for lab assessments: align hostname, OS banner strings, and scanner user agents. Editing /etc/os-release incorrectly can break package management — change only what you understand.

## Commands

Match hostname in:
    /etc/hostname
    /etc/hosts

Banner strings (lab use only):
    /etc/issue
    /etc/issue.net
    /etc/os-release

hostnamectl

Firefox about:config (examples):
    general.platform.override > Win32
    general.useragent.override > <current Firefox UA>
    privacy.resistFingerprinting > true
    media.peerconnection.enabled > false

Scanner user agents:
    Discover Update refreshes resource/user-agent.txt (Edge) and patches Nikto, Nmap, and ffuf.
    Manual reference: https://www.whatismybrowser.com/guides/the-latest-user-agent/edge
    Inspect: cat $DISCOVER/resource/user-agent.txt
    ffuf config: ~/.config/ffuf/ffufrc

## Validation / False Positives

Cloaking UA strings reduces scanner fingerprinting — it does not hide network-level TLS or behavior fingerprints.
